# YMCA-Membership-Data-Analytics-Project-
YMCA Active Members Analysis: 2022 vs 2026

A Python/Pandas data analysis project comparing YMCA Greenfield's active membership data across 2022 and 2026. Built for Google Colab, the notebook cleans raw membership export CSVs and generates visualizations and summary statistics covering:

- Membership type distribution
- Age demographics and age bands
- Discount usage by membership type and age group
- Payment method breakdown
- Monthly billing and revenue trends
- Household size distribution
- Year-over-year comparison of key metrics (total members, average age, discount rates, revenue, etc.)

The project standardizes messy raw exports (multi-row headers, embedded age strings in DOB fields, inconsistent discount/gender fields) into clean, analysis-ready datasets, then exports a combined CSV for further use.

Tech stack: Python, pandas, NumPy, matplotlib, seaborn, Google Colab
